This webtool helps you to encrypt text message over an untrust network/communication channels.
Just type your plain text/message in the first text box. (use Clear text button to clear text box)

As of the password or SymKey, you can either
  a) key-in you own password to the box near SymKey label;
  b) Press GenKey button to generate a random key

Since the generate random key can be hard to note down, you may use Key2QR button
to convert the GenKey into a QR code that can be read by any QR code reader.
